# twitter-util
twitterの操作を自動化するコマンドラインツール

## Functions
* ハッシュタグをつぶやいたユーザーを自動でフォロー（大型勉強会の時などにどうぞ）
* フォロー返し
* 相互フォロー以外をリムーブ

## usage
* git clone
* eclipseで読み込む
* token.default.propertiesをtoken.propertiesにリネーム、自分のトークンを入力
* Mainクラスのoperationからやりたい操作を選ぶ
* 実行
